## Introducing Project
<p>
  DRINKABLE은 다양한 칵테일을 소개하는 웹 프로젝트입니다. 사용자는 원하는 칵테일 정보와 필요한 재료를 쉽게 확인할 수 있습니다.<br/>
  더불어, 맛, 양, 논알콜 여부에 따라 칵테일을 효과적으로 분류할 수 있어, 사용자는 자신의 기분에 맞는 칵테일을 쉽게 선택할 수 있습니다.<br/>
  이는 프로젝트의 주요 강점 중 하나로 강조될 것입니다.<br/><br/>
</p>
