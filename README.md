## Introducing Our Project
DRINKABLE은 다양한 칵테일을 소개하는 웹 프로젝트입니다. 사용자는 원하는 칵테일에 대한 정보와 필요한 재료를 쉽게 알 수 있습니다.
뿐만 아니라  맛, 양, 논알콜 여부에 따라 칵테일을 분류할 수 있어, 사용자는 자신의 기분에 따라 알맞은 칵테일을 선택할 수 있습니다. 이는 프로젝트의 큰 장점으로 강조될 것입니다.

<br><br>

## Getting Started

### 1. Client
    cd client
    npm i
    npm run start
    
### 2. Server
    cd server
    npm i
    nodemon server
